Score lock-ins

Use this for prediction receipts before matches. Each player can lock at least two scores per day using Match Slot 1 and Match Slot 2. Locked scores are shown on the board and cannot be removed for 20 hours in this browser.


Lock rule

Lock period: 20 hours after submission.

Locked scores are visible immediately on the lock-in board. On a normal static GitHub Pages site, browser lock-ins do not automatically sync across everyone’s phones. For everyone to submit from their own phone and have it instantly visible to all, this would need a shared backend such as Google Forms/Sheets, Firebase, or Supabase.

Scoring note

Right now, 1 point = 1 correct exact score. Same goal-difference hits are manual close-score stats, like predicting 2-1 when the real score was 1-0. Ranking order is: points first, then goal-difference hits, then more predictions made, then alphabetical.

Projection setup: change Games left in the Update tab whenever the tournament moves forward. Projected points and possible predictions will follow that number.
