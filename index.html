Exact fixtures & actual scores

Pick a FIFA fixture, enter actual score when the game is done, and it displays on the board. Once an actual score is saved, there is no remove button on the site.


Player prediction

Predictions are tied to exact fixtures. If an actual score exists, points and GD hits calculate automatically. You can still add manual adjustments in the Update tab.

What auto-calculates

Predictions made: counted from fixture picks + manual extra predictions.

Points: exact score = 1 point + manual extra points.

GD hits: same goal difference = 1 GD hit + manual extra GD hits.

Static-site note: GitHub Pages cannot securely fetch/update live FIFA scores for everyone without a backend. This page is built with the current fixture list and lets you manually enter actual scores after matches.

Manual update / adjustments

Use this for older predictions, corrections, or anything not entered under Fixtures & Picks. Totals = fixture-calculated stats + these manual adjustments.
